# **App Name**: Umrah Journey

## Core Features:

- Homepage Showcase: Display key information on a single, scrollable page, including featured packages, itinerary highlights, facilities, photo gallery, and testimonials.
- Informational Pages: Provide separate pages for 'About Us' (company profile) and 'Requirements', accessible via the main navigation menu.
- Floating WhatsApp Button: Implement a permanently visible WhatsApp button for immediate inquiries and registration.
- Professional Footer: Include complete contact information (address, email, phone) and other important links in the website footer.
- Responsive Design: Ensure the website adapts seamlessly to desktop, tablet, and mobile devices.
- Dynamic Content: Offer CMS capabilities for managing and updating website content. Owners will be able to add/modify travel packages, update pricing, manage images and add testimonials.

## Style Guidelines:

- Primary color: Dark slate blue (#3F51B5) for a trustworthy and professional feel.
- Background color: Very light grey (#F0F0F5), nearly white, provides a clean, modern backdrop.
- Accent color: A subdued purple (#9575CD), a shade brighter and more saturated than the primary, will highlight key interactive elements.
- Headline font: 'Playfair', serif, for an elegant and fashionable touch on titles; body font: 'PT Sans', sans-serif, for clear readability of content.
- Use modern and minimalist icons related to travel and Islamic themes.
- Inspired by 'Japan Tour' and 'nakhla.id', prioritize large, high-quality photos and a well-organized information structure.
- Use smooth, lightweight transitions like fade-in/out, slide-in, and parallax scrolling to create a dynamic but performant user experience. Optimize animations for performance to ensure fast loading times.
